11. What is the difference between HTML and XHTML?

HTML stands for Hypertext Markup Language, whereas XHTML stands for Extensible Hypertext Markup Language. XHTML is basically an extension of HTML, which is stricter than HTML. 12. How many HTML versions are available in the market?

13. What is DOCTYPE?

A document type declaration, or DOCTYPE, is an instruction that associates a particular XML or SGML document (for example, a web page) with a document type definition (DTD) (for example, the formal definition of a particular version of HTML 2.0 - 4.0)

14.Explain HTML structure?

Document Type Declaration (<!DOCTYPE>): This declaration specifies the HTML version being used and helps browsers interpret the document correctly.
HTML Element (<html>): This is the root element and contains the entire HTML document.
Head Element (<head>): This element provides meta-information about the document, such as the title, character encoding, and references to external stylesheets or scripts. It does not represent visible content.
Body Element (<body>): This element encloses the visible content of the web page, including text, images, links, and other elements.

15.What is HTML? How are HTML tags formed? Explain HTML comments?

HTML (Hypertext Markup Language) is a standard markup language used for creating web pages. It provides a structure and format for organizing and presenting content on the internet.
HTML tags are formed by using angular brackets to enclose the tag name. HTML comments provide a way to add notes or explanations within the HTML code that are not displayed on the web page.HTML comments are written between <!-- and --> and can span multiple lines. The comment text is not rendered by the browser and is only visible in the source code.

16.What is the use of Anchor tag? Explain two ways to specify URL in href attribute?

The anchor tag <a> is used for creating hyperlinks in HTML. It allows users to navigate to other web pages or resources. The `href` attribute of the <a> tag specifies the URL of the destination.
There are two ways to specify the URL:
1.Absolute URL:It includes the complete address of the resource, including the protocol (e.g., "http://" or "https://") and the domain name. It is used for linking to external web pages or resources on different websites.
Example: <a href="https://www.example.com">Visit Example </a>
2. Relative URL:It specifies the path or location of the resource relative to the current document. It is used for linking to internal pages within the same website or referencing resources within the same directory or folder structure.
Example: <a href="/about.html">About </a>

To create a hyperlink in HTML, you can use the anchor tag <a>. The anchor tag is specifically designed for creating links.

23. Which html tags are used to create table structure? Explain in brief at least five

An HTML table consists of one <table> element and one or more <tr>, <th>, and <td> elements. The <tr> element defines a table row, the <th> element defines a table header, and the <td> element defines a table cell. An HTML table may also include <caption>, <colgroup>, <thead>, <tfoot>, and <tbody> elements.

24. What is a caption?

The <caption> tag defines a table caption. The <caption> tag must be inserted immediately after the <table> tag. Tip: By default, a table caption will be center-aligned above a table. However, the CSS properties text-align and caption-side can be used to align and place the caption.

25. Which attribute allows you to add descriptive text to a table?

The attribute that allows you to add descriptive text to a table in HTML is the <caption> element. The <caption> element is used to provide a title or description for a table, making it more accessible and informative for users.

26. Which HTML attribute is used to set the space between cells in an HTML element?

The HTML <table> cellspacing Attribute is used to specify the space between the cells. The cellspacing attribute is set in terms of pixels. Attribute Values: pixels: It sets the space between the cells in terms of pixels.

27. Which attribute do you use to span table cells across tablecolumns?

The colspan attribute defines the number of columns a table cell should span.

34. What are the Input types in HTML?

3. number for numeric input.
4. email for email addresses.
5. tel for telephone numbers.

7. date for selecting a date.

35. What are new Input types included in HTML5 ?

Some of the new input types included in HTML5 are:

1. color Presents a color picker to select a color.
2. date Displays a date picker to choose a specific date.
3. datetime Allows input for a specific date and time, including timezone information.
4. datetime-local Enables input for a date and time without the timezone.
5. email Specifically designed for capturing email addresses.
6. month Shows a month and year picker for selecting a specific month.
7. number Allows users to enter numeric values, including options for min, max, and step attributes.
8. range Represents a slider control for selecting a value within a specified range.
9. search Provides a text input field for search queries, often with search-specific styling.
10. tel Used for capturing telephone numbers.
11. time Displays a time picker to choose a specific time.
12. url Intended for capturing website URLs.
13. week Shows a week picker for selecting a specific week of the year.

These new input types introduced in HTML5 provide more specific and specialized input options, improving the user experience and enabling better data validation.

36. What is the use of <fieldset>?

The <fieldset> element in HTML is used to group related form elements together. It provides a way to semantically organize and label a set of form controls within a form.

37. What is the use of <nav> tag?

The <nav> element is used to define a section of a web page that contains navigation links. It is specifically designed for marking up the primary navigation section of a website. By using the <nav> element, you can group navigation-related links together and provide a clear structure for navigation. This improves accessibility, search engine optimization, and the overall understanding of the page's structure.

38. Difference between <span> and <div> tag?

<span> is used to apply styles to specific portions of text or inline elements within a block of content. It is an inline element and does not affect the layout.
<div> is used to group and create divisions within an HTML document. It is a block-level element and can contain other elements. It is commonly used for styling or layout purposes.

39. How to embed Javascript in HTML ?

JavaScript code can be embedded inline into an HTML document by using the dedicated HTML tag "script". This HTML tag wraps around the JS code. The "script" tag can be placed either in the "head" section of your HTML or in the "body" section. The placement entirely depends on when you want the JS code to load.

40. How do we specify line breaks in HTML?

To do a line break in HTML, use the "br" tag. Simply place the tag wherever you want to force a line break. Since an HTML line break is an empty element, there's no closing tag.

41. Explain HTML 5 Element ?

Represents a piece of content that is only slightly related to the rest of the page. Defines an audio file. This is used for rendering dynamic bitmap graphics on the fly, such as graphs or games.

42. Which tag is used in HTML5 to display video ?

<video> tag is used in HTML5 to display videos on web pages.

43. HTML Event Attributes on unload fires

The on unload event attribute fires when a user is about to leave a web page or close the browser window/tab. It is commonly used for cleanup tasks or displaying a confirmation message.
